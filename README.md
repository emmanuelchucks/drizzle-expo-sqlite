## Features

- Drizzle ORM
- Expo SQLite
- Expo SDK 50
- Expo Router v3
- Nativewind v4

## Run locally

If you already have a simulator/emulator set up, this should open Expo Go automatically:

```zsh
bun install
bun run ios # or android
```
